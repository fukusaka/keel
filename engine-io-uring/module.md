# Module engine-io-uring

Linux io_uring-based IoEngine implementation with multishot accept/recv and provided buffer ring.

# Package io.github.fukusaka.keel.engine.iouring

IoUringEngine, IoUringChannel, IoUringEventLoop, and pipeline support (IoUringPipelinedChannel, IoUringIoTransport). Includes IoUringCapabilities for feature probing and graceful fallback.
