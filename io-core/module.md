# Module io-core

Buffer primitives and async I/O abstractions for all keel engines.

# Package io.github.fukusaka.keel.buf

IoBuf (reference-counted buffer), BufferAllocator, SlabAllocator, and platform-specific implementations (NativeIoBuf, DirectIoBuf, TypedArrayIoBuf).

# Package io.github.fukusaka.keel.io

SuspendSource / SuspendSink async I/O interfaces, BufferedSuspendSource / BufferedSuspendSink with line-oriented parsing and deferred flush.
